# КавказТорг 07 - Сайт-визитка

Современный адаптивный сайт для компании "КавказТорг 07" с админ-панелью для управления товарами.

## Файлы проекта

- `index.html` - Главная страница сайта
- `admin.html` - Админ-панель для управления товарами
- `styles.css` - Стили сайта
- `script.js` - JavaScript для главной страницы
- `data.js` - Система управления данными товаров (localStorage)
- `logo.png.PNG` - Логотип компании

## Функционал

### Главная страница
- Адаптивный дизайн
- Интерактивный фон с горами Кавказа
- Каталог товаров (поддоны и напитки)
- Информация о компании
- Контактные данные

### Админ-панель
- Добавление товаров в категории
- Установка цены, описания, изображений
- **Загрузка изображений напрямую из файлов** (JPG, PNG, GIF, WebP)
- Автоматическая оптимизация изображений (сжатие до 2 МБ)
- Превью изображений перед добавлением
- Выбор категории (поддоны/напитки)
- Удаление товаров
- Изменения сохраняются в localStorage и отображаются на сайте в реальном времени

## Доступ к админ-панели

Админ-панель доступна по ссылке: `admin.html` (также есть ссылка в футере главной страницы)

## Начальные данные

При первом запуске автоматически создаются:
- Европаллет (европейский) - 500 ₽
- Coca-Cola - 80 ₽
- Sprite - 75 ₽
- Fanta - 75 ₽
